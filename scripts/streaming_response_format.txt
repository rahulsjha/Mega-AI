STREAMING RESPONSE FORMAT

Every SSE message is sent as:

data: {JSON_OBJECT}\n\n

Expected TRACE_EVENT JSON fields:
- timestamp_utc: ISO 8601 timestamp
- job_id: UUID string for the request/job
- agent_name: orchestrator, decomposition, rag, critique, synthesis, compression, or tool/agent identifier
- event_type: AGENT_START, TOOL_CALL, TOOL_RESULT, AGENT_END, or ERROR
- tool_name: optional tool name for tool events
- tool_input: input object for tool call events
- tool_output: output object for tool result/end events
- token_cost_this_step: integer token count for the current step
- cumulative_token_cost: total tokens used so far
- budget_remaining_pct: remaining budget percentage
- shared_context_snapshot: snapshot of current agent context
- error: null on success, or error string on failure

Example stream:

data: {"timestamp_utc":"2026-05-11T00:00:00.000Z","job_id":"<uuid>","agent_name":"orchestrator","event_type":"AGENT_START","tool_name":null,"tool_input":null,"tool_output":null,"token_cost_this_step":0,"cumulative_token_cost":0,"budget_remaining_pct":100.0,"shared_context_snapshot":{"query":"..."},"error":null}

data: {"timestamp_utc":"2026-05-11T00:00:01.000Z","job_id":"<uuid>","agent_name":"decomposition","event_type":"AGENT_START","tool_name":null,"tool_input":null,"tool_output":null,"token_cost_this_step":0,"cumulative_token_cost":0,"budget_remaining_pct":95.0,"shared_context_snapshot":{},"error":null}

data: {"timestamp_utc":"2026-05-11T00:00:02.000Z","job_id":"<uuid>","agent_name":"decomposition","event_type":"AGENT_END","tool_name":null,"tool_input":null,"tool_output":{"final_answer":"..."},"token_cost_this_step":120,"cumulative_token_cost":120,"budget_remaining_pct":94.0,"shared_context_snapshot":{},"error":null}

data: {"timestamp_utc":"2026-05-11T00:00:03.000Z","job_id":"<uuid>","agent_name":"orchestrator","event_type":"AGENT_END","tool_name":null,"tool_input":null,"tool_output":{"final_answer":"...","total_latency_ms":1234.5},"token_cost_this_step":0,"cumulative_token_cost":120,"budget_remaining_pct":94.0,"shared_context_snapshot":{},"error":null}

If a failure happens, the stream emits:

data: {"timestamp_utc":"...","job_id":"<uuid>","agent_name":"orchestrator","event_type":"ERROR","tool_name":null,"tool_input":null,"tool_output":null,"token_cost_this_step":0,"cumulative_token_cost":0,"budget_remaining_pct":0.0,"shared_context_snapshot":{},"error":"<message>"}

Run the shell test runner:

bash scripts/run_stream_test.sh

The script will:
1. Install Python dependencies if requirements.txt exists.
2. Start the backend with SQLite.
3. POST to /query with Accept: text/event-stream.
4. Capture the streamed TRACE_EVENT output.
5. Print the output plus an example format reference.
